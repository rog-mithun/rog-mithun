🎓 I'm a final-year MSc AI & ML student passionate about building solutions using AI, cloud, optimization, and embedded systems.  
🚀 I work at the intersection of **ML, IoT, full-stack, cloud** & **analytics**.

## 🚀 Highlight Projects

### 🧠 SkillMorph AI  
LLM-powered simulator that generates personalized career paths using Gemini + SadTalker.

### 🤖 RL Chatbot Assistant  
Multi-turn voice-based chatbot trained via Reinforcement Learning (DQN + BERT).

### ☀️ Solar PSO Optimizer  
Selects optimal locations for solar plants using Particle Swarm Optimization + FastAPI + React.

### 🧵 IoT Textile Monitor  
Embedded system with BH1750 + ESP32 to monitor idle time in textile industries.
